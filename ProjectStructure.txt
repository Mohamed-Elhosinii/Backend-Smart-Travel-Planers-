================================================================================
  SMART TRAVEL PLANERS - PROJECT STRUCTURE
  Solution: Backend (Smart Travel Planers).slnx
================================================================================

The solution follows a 3-layer architecture:
  - DAL  (Data Access Layer)   : Database entities, repositories, migrations
  - BLL  (Business Logic Layer): Services, DTOs, external APIs, AI agents
  - PL   (Presentation Layer)  : Controllers, entry point, configuration

================================================================================
  ARCHITECTURE SUMMARY
================================================================================

  PL (Presentation)
   |  Controllers expose REST API endpoints
   |  Program.cs wires up all dependencies
   |
   v
  BLL (Business Logic)
   |  Services contain the core logic (Auth, Chat, Email)
   |  DTOs define data shapes for API input/output
   |  AI/ contains Semantic Kernel agents and plugins
   |  ExternalApis/ wraps 3rd-party HTTP integrations:
   |     - Foursquare (Places)
   |     - Serper     (Google Search)
   |     - StayAPI    (Hotels)
   |     - Flight API (Flights)
   |     - Visual Crossing (Weather)
   |
   v
  DAL (Data Access)
     Entities map to SQL Server tables
     Repositories handle CRUD via EF Core
     UnitOfWork coordinates transactions
     Migrations version-control the DB schema

================================================================================
  KEY TECHNOLOGIES
================================================================================

  - ASP.NET Core Web API        : HTTP server and REST API framework
  - Entity Framework Core       : ORM for SQL Server database
  - ASP.NET Core Identity       : User management and authentication
  - JWT Bearer Tokens           : Stateless API authentication
  - Google OAuth 2.0            : Social login via Google
  - Microsoft Semantic Kernel   : AI orchestration framework
  - OpenAI GPT-4o-mini          : Language model for AI chat and agents
  - Foursquare Places API       : Place/POI search
  - Serper API                  : Google Search integration
  - StayAPI                     : Hotel search
  - Visual Crossing Weather API : Weather forecasts
  - SMTP (Gmail)                : Email sending
